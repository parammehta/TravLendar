Instructions to set up a local https server
1. Download nodejs
2. Check on cmd if 'node' and 'npm' command work
3. Save openssl.cnf in the directory that has Travlendar
4. Open cmd as an admin and go to that directory
5. Execute: openssl req -x509 -sha256 -nodes -days 365 -newkey rsa:2048 -keyout key.pem -out cert.pem -config=openssl.cnf
	if the -config parameter does not work:
	5.1 Execute: set OPENSSL_CONF=<path_to_openssl.cnf>
	5.2 Execute: openssl req -x509 -sha256 -nodes -days 365 -newkey rsa:2048 -keyout key.pem -out cert.pem
6. Make sure you have key.pem and cert.pem in the directory with openssl.cnf

Directory structure:
.....\MyDirectory\Travlendar\www\....
.....\MyDirectory\openssl.cnf
.....\MyDirectory\key.pem
.....\MyDirectory\cert.pem

7. Install http-server:
Run: npm install http-server -g
8. Run: http-server Travlendar\www -p 4443 -S
9. Go to https:\\localhost:4443
